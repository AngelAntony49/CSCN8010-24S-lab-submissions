This is the repository to which the CSCN8010 practical labs are submitted to.

Setup
Fork this repository (this is critical, as it will allow you to push your submissions to Github)
Clone the forked repository to your local computer (git clone REPO-URL, where REPO-URL is the URL of the forked repository)
Follow the instruction in Student Submission Subfolder Setup
Commit and push your updates to your forked repository on Github
Note, you can use the same environment as CSCN8010, so no need to create another one
